Instructions

How to test the accessibility features:

  1. Look for the ♿ icon in the top-right corner of this page
  2. Click the icon to open the accessibility menu
  3. Try different features like:
    • Text Size Adjustment - fine-grained control (0-150%)
    • Grayscale Mode - convert to grayscale for better focus
    • High Contrast - enhance color contrast
    • Negative Contrast - invert colors for reduced eye strain
    • Light Background - force light backgrounds
    • Dark Mode - switches to dark theme
    • Links Underline - make all links clearly underlined
    • Readable Font - apply clean, readable fonts
    • Focus Indicators - better focus outlines
    • Reduce Motion - minimizes animations
    • Reset All - quickly return to default settings
  4. Notice how each feature affects the appearance of this page
  5. Your preferences are saved and will persist across page refreshes
